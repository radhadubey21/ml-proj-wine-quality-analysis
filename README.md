# 🍷 Wine Quality Prediction (Machine Learning Project)

## 📌 Overview

This project focuses on predicting the quality of wine using Machine Learning techniques.
It involves data analysis, visualization, and model building using Python libraries.

* Dataset used: **Wine Quality Dataset**
* File: `winequality-red.csv`
* Features include:

  * pH
  * Sulphates
  * Acidity
  * etc.
